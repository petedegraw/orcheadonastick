# Orc Head

A simple single-page website displaying the iconic orc head on a stick from Lord of the Rings: The Two Towers.

## Structure

```
orchead/
├── index.html      # Main HTML page with SEO meta tags
├── styles.css      # Stylesheet
├── assets/         # Image assets
└── README.md
```

## Features

- Responsive image display (centered, max-height 100%)
- Bright orange background
- SEO optimized with Open Graph and Twitter card meta tags

## Usage

Open `index.html` in a browser.
